Bruno5p Repository

Personal Kodi addons — anime tools, MAL integration & media routing

How to Install

  1. In Kodi, go to Settings → System → Add-ons and enable Unknown sources.
  2. Download the repository zip:
    repository.bruno5p-1.0.0.zip
  3. In Kodi go to Add-ons → Install from zip file and select the downloaded file.
  4. Once installed, go to Install from repository → Bruno5p Repository and install any addon.

Available Add-ons

  • Anime Scraper (MyAnimeList)
    metadata.anime.mal
    Fetches anime metadata from MyAnimeList via the Jikan API. Movies, OVAs and specials appear as Season 0.
    v1.0.0
  • List Builder
    plugin.list.builder
    Build and browse curated video lists from TMDb Discover filters or dynamic samplers from Kodi smart playlists.
    v1.1.0
  • Anime Library Manager
    script.anime.library
    Rebuilds the anime symlink tree and triggers a Kodi library scan so nested anime collections are properly indexed.
    v1.0.0
  • MAL Manager
    script.mal.manager
    Authenticate with MyAnimeList and manage your anime watchlist directly from the Kodi context menu.
    v1.3.0
  • Media Router
    script.media.router
    Routes anime, TV shows and movies to local playback or streaming addons based on configurable per-type settings.
    v1.1.0
